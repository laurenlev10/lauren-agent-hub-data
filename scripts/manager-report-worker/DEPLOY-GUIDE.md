# שמירה אוטומטית לטופס המנהלת — מדריך הפעלה (10 דקות, חד-פעמי)

הטופס כבר עובד **בלי** השלב הזה — המנהלת ממלאת ולוחצת "שליחה לקבוצה",
וזה נשלח בוואטסאפ עם התמונות. השלב הזה מוסיף **שמירה אוטומטית**: כל דוח
שהמנהלת שולחת נשמר לבד ומופיע אצלך בדשבורד (`?mode=inbox`) — בלי שאת צריכה
לעשות כלום, ובלי שלמנהלת יש גישה לגיטהאב.

איך זה עובד: שרת זעיר וחינמי ב-Cloudflare ("Worker") מחזיק את ה-token של
גיטהאב **בצד השרת**, ולכן אף פעם לא נחשף בטלפון של המנהלת. הטופס שולח אליו
את הדוח, והוא שומר אותו ברסיטורי.

---

## שלב 1 — צרי Worker ב-Cloudflare
1. היכנסי ל-https://dash.cloudflare.com → בתפריט: **Workers & Pages** → **Create** → **Create Worker**.
2. תני שם, למשל `mbs-manager-report` → **Deploy** (תקבלי Worker ריק זמני).
3. לחצי **Edit code**. מחקי את כל הקוד שיש שם, והדביקי את כל התוכן של הקובץ
   **`manager-report-worker.js`** (מצורף). → **Deploy**.

## שלב 2 — הוסיפי את ה-token כסוד (Secret)
1. בעמוד ה-Worker → **Settings** → **Variables and Secrets** → **Add**.
2. סוג: **Secret**. שם (Name) — חשוב, בדיוק כך: **`GH_TOKEN`**.
3. ערך (Value): ה-GitHub token שיש לך (אותו אחד עם הרשאת **Contents: write**
   על `laurenlev10/lauren-agent-hub-data`). → **Save and deploy**.

## שלב 3 — העתיקי את כתובת ה-Worker
בראש עמוד ה-Worker מופיעה כתובת כמו:
`https://mbs-manager-report.<something>.workers.dev`
העתיקי אותה.

## שלב 4 — הדביקי את הכתובת בקובץ הקונפיג
פתחי את `docs/manager-report/config.json` ברסיטורי ושימי שם את הכתובת:
```json
{ "save_endpoint": "https://mbs-manager-report.<something>.workers.dev" }
```
שמרי (commit + push). זהו — מאותו רגע **כל** מנהלת, מכל טלפון, שולחת דוח שנשמר
אוטומטית. אין צורך להגדיר כלום בטלפון של אף אחת.

> רוצה שאני אעשה את שלב 4 בשבילך? פשוט שלחי לי את כתובת ה-Worker ואני אכניס
> אותה ל-config.json ואדחוף — זהו.

## איך לבדוק שזה עובד
1. פתחי טופס של אירוע כלשהו (כפתור **📋 דוח מנהלת** בדשבורד), מלאי משהו, לחצי
   **שליחה**. אמור להופיע "✓ Saved".
2. פתחי `https://dashboard.themakeupblowout.com/manager-report/?mode=inbox` —
   הדוח אמור להופיע ברשימה.

## אבטחה — כמה זה בטוח?
- ה-token נמצא רק בצד השרת (Secret ב-Cloudflare). הוא **לא** בדף, ולא בטלפון של המנהלת.
- ה-Worker מקבל רק POST ושומר דוח. גם אם מישהו ינחש את הכתובת, המקסימום שהוא
  יכול לעשות זה להוסיף דוח — לא למחוק מלאי, לא לגעת בשום דבר אחר.
- רוצה הגבלה נוספת? אפשר להגביל את ה-Worker לקבל רק מ-`dashboard.themakeupblowout.com`
  (בקוד: לשנות את `Access-Control-Allow-Origin` לדומיין שלך). תגידי ואני אעשה.

## בלי השלב הזה (ברירת המחדל)
הטופס עדיין שלם ועובד: המנהלת ממלאת → "שליחה לקבוצה" פותח את שיתוף הוואטסאפ
עם הדוח והתמונות → היא שולחת לקבוצה. בנוסף יש "📋 העתקת טקסט", "🔗 העתקת לינק",
ו-"🖨️ הדפסה/PDF". כשאת פותחת לינק של דוח מהטלפון שלך (עם ה-token), הוא נשמר
אצלך בדשבורד אוטומטית.
