Recipient list patterns:
• {event_year_list_name} — only the new list created this year (e.g. Tulsa, OK 2026)
• match:{city} — every list whose name contains the city name (any year, any format)

• Combine with commas: {event_year_list_name}, match:{city} = all lists with the city name from any year (deduped)
