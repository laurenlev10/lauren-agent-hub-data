🎚️ TIER — תקציבים והגדרות לכל רמת אירוע

מקור אמת יחיד: docs/state/event_radius.json. כל שינוי שנשמר כאן מתעדכן אוטומטית בכל הדשבורדים והסוכנים.

מה זה Tier?

כל אירוע מקבל דירוג 1–3 לפי גודל/פוטנציאל העיר. ה-Tier קובע: כמה עובדים באירוע, תקציב שיווק ל-4 שבועות (וההתפלגות בין Meta ל-TikTok), תקציב שכירות אולם ותקציב הזמנת סחורה. Tier 1 = האירועים הגדולים, Tier 3 = הקטנים.
