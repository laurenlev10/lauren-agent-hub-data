🎯 מה הסוכן יבדוק

  • 🏨המקום הנוכחי — האם המקום עומד בסטנדרט MBS? sqft ≥2,500, mid-tier brand, חניה חינם, load-in ישיר (בלי מעלית)
  • 📊דמוגרפיה ZIP — Hispanic% (יעד 25-50%), median income ($55-75K), % נשים 25-44 (30%+), Walk Score (50+)
  • 🏙️סריקת העיר — 3-5 שכונות עם הכי הרבה foot traffic, אחוז Hispanic גבוה, וקרבה למרכזי קניות
  • 🏆5-10 venues חלופיים בעיר — Hilton Garden Inn / Holiday Inn / DoubleTree / Embassy Suites / Sheraton / Best Western Premier (לא: Hilton מלא, Marriott מלא, Westin, Ritz)
  • 📅היסטוריית MBS בעיר — תוצאות עבר מ-historical xlsx + סיבות לחולשה (Covid? מזג אוויר? אולם לא נכון?)
  • 🌆ערים סמוכות שכדאי לבדוק באותה נסיעה — Cross-city ranking לפי MBS optimal profile score (X/30)
  • 📋Contract criteria checklist להדפסה — sqft, חניה, load-in, capacity, AV, no catering minimum

📦 מה יחזור בסיום

  • דוח HTML מלא ב-Hebrew RTL ב-docs/scout/<city-slug>/
  • סעיף פותח "הערכת המקום הנוכחי" — 3 חוזקות + 3 חולשות של האולם
  • Cross-city ranking table (העיר + ערים סמוכות, מהטוב לחלש)
  • סעיף סיום "המלצה סופית" עם verdict ברור: 🟢 להישאר / 🟡 לבדוק חלופה / 🔴 להחליף בהזדמנות הבאה
  • 3+ venues חלופיים עם ציון, כתובת, contact, מחיר משוער
  • SMS דרך SimpleTexting עם הציון והקישור
